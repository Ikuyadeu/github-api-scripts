# ohmyzsh-bot

> A GitHub App built with [Probot](https://github.com/probot/probot) that A bot to manage the ohmyzsh project

## Setup

```sh
# Install dependencies
npm install

# Run typescript
npm run build

# Run the bot
npm start
```
