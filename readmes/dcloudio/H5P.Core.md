# H5P.Core
HTML5plus Runtime,参考:[HTML5+](http://html5plus.org/)简称5+Runtime，运行于手机端的强化web引擎，除了支持标准HTML5外，还支持更多扩展的js api。
H5P.Core 为5+Runtime核心组件定义并实现了一套插件机制，可以基于该工程进行自己插件的开发和调试。
开发的插件可以集成到5+Runtime中使用。
